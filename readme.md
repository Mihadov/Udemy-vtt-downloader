# Udemy Subtitle (VTT) Downloader

Браузерное расширение (MV3) для быстрого и удобного скачивания всех субтитров (файлов `.vtt` / JSON) с любого курса Udemy в один клик. Идеально подходит для локального просмотра, перевода и конспектирования.

---

## 🚀 Возможности
- **Два формата экспорта (Новое в v1.1):** Сохраняйте лекции как чистый JSON для разработчиков или в красивом структурированном формате Markdown (`.md`) для баз знаний (Notion, Obsidian).
- **Поддержка корпоративных аккаунтов (Новое в v1.1):** Полноценная работа с поддоменами Udemy Business (например, `company.udemy.com`).
- **Пакетное скачивание:** Нажмите одну кнопку, и расширение автоматически спарсит весь курс и скачает все доступные субтитры (English [Auto]).
- **Современный UI:** Элегантный переключатель форматов и визуальный прогресс-бар процесса скачивания.
- **Работает локально:** Без сторонних серверов. Расширение напрямую работает с GraphQL API Udemy через ваш браузер, гарантируя 100% безопасность и приватность ваших данных.
- **Устойчивость к изменениям Udemy:** Использует 6 алгоритмов поиска (fallback) ID курса внутри страницы (мета-теги, `udemy_com:course_id`, `data-module-args`, `og:image`), чтобы всегда стабильно находить нужные данные.

## 📦 Установка (Для пользователей)

Если вы просто хотите скачать готовое расширение:

1. Перейдите на вкладку [Releases](../../releases) *(замените ссылку при публикации в GitHub!)* и скачайте последний файл **`UdemyVTTDownloader-v1.1.zip`**.
2. Распакуйте архив в любую удобную папку на компьютере.
3. Откройте Google Chrome (или Яндекс.Браузер / Edge) и перейдите по адресу: `chrome://extensions/`.
4. Включите **Режим разработчика** (Developer mode) в правом верхнем углу.
5. Нажмите кнопку **Загрузить распакованное расширение** (Load unpacked) в верхнем левом углу.
6. Выберите распакованную папку.
7. Иконка расширения появится на панели браузера. Закрепите её для удобства!

## 🛠 Как выпустить релиз (Для разработчика)

Чтобы создать готовый загрузочный файл для GitHub Releases (упаковка расширения):

### Способ 1: Вручную
Просто выделите _содержимое_ папки `extension/` (файлы `manifest.json`, `background.js` и т.д.) и добавьте их в ZIP-архив (например, `UdemyVTTDownloader-v1.1.zip`).
Внимание: архивировать нужно именно файлы внутри папки `extension`, а не саму папку `extension`, чтобы `manifest.json` лежал в корне архива!

### Способ 2: Через команду в Windows (PowerShell)
Вы можете быстро собрать релизный архив с помощью команды терминала из корневой папки проекта:
```powershell
Compress-Archive -Path "extension\*" -DestinationPath "UdemyVTTDownloader-v1.1.zip" -Force
```
*Эта команда создаст готовый `UdemyVTTDownloader-v1.1.zip`, который сразу можно загружать на GitHub!*

## 👨‍💻 Как работает под капотом

Если вам интересно, как здесь всё устроено:

1. **Popup (`popup.html` / `popup.js`)**: Ваш интерфейс. При нажатии на кнопку он отправляет сигнал в окно самого курса.
2. **Content Script (`content.js`)**: Внедряется на страницу Udemy. Его цель — любыми путями достать `courseId` из закрытого кода страницы, используя многоуровневый парсинг.
3. **Background Worker (`background.js`)**: Ядро приложения. Получив `courseId`, фоне делает запрос к GraphQL API Udemy авторизованной сессией вашего браузера, вытягивает Curriculum (список всех видео), скачивает каждый `.vtt` файл по отдельности, затем объединяет их и отправляет браузеру команду на сохранение финального файла на ваш диск.
