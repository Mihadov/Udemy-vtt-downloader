# Udemy Subtitle (VTT) Downloader

**Udemy Subtitle Downloader** — это браузерное расширение (MV3), созданное для легкого, быстрого и пакетного скачивания субтитров (файлов `.vtt`) ко всем видеокурсам на платформе Udemy.

Расширение отличается простотой в использовании: вы нажимаете одну кнопку, видите визуальный прогресс-бар, а в конце получаете единый файл со всеми скачанными субтитрами.

---

## ⚡️ Как это работает (Архитектура)

Расширение состоит из нескольких ключевых компонентов, взаимодействующих друг с другом:

### 1. **Popup (Интерфейс пользователя)**
- `popup.html` / `popup.js` / `popup.css`
- **Роль:** Отвечает за взаимодействие с пользователем. При клике на иконку расширения открывается минималистичный интерфейс.
- **Особенности:**
  - Имеет кнопку загрузки, которая инициализирует весь процесс.
  - Содержит анимированный прогресс-бар для визуализации процесса скачивания (сколько лекций обработано из общего числа).
  - При возникновении ошибок или успешном завершении выводит понятные статусные сообщения пользователю.

### 2. **Content Script (Анализатор страницы)**
- `content.js`
- **Роль:** Внедряется непосредственно на страницу Udemy, открытую в вашем браузере.
- **Особенности:**
  - Главная задача — **извлечь уникальный идентификатор курса (Course ID)**.
  - Поскольку структура страниц Udemy часто меняется, скрипт использует 6 каскадных (fallback) методов поиска Course ID. Он ищет его в мета-тегах (`udemy_com:course_id`, `og:image`), извлекает из специальных DOM-атрибутов (`data-module-args`, `data-clp-course-id`) и наконец парсит из JSON-состояний внутри `<script>` тегов.
  - Найденный Course ID передается дальше в фоновый скрипт.

### 3. **Background Script (Фоновый процесс и работа с API)**
- `background.js`
- **Роль:** Это "сердце" расширения, которое работает в фоновом режиме по модели Service Worker (Manifest V3). Выполняет всю тяжелую работу по API запросам, так как имеет доступ к куки авторизации пользователя на Udemy.
- **Что происходит внутри:**
  1. Выполняет запрос к **GraphQL API Udemy** по найденному Course ID, чтобы получить всю учебную программу (Curriculum) курса (разделы и лекции).
  2. Ищет среди всех лекций (тип `Video`) те, к которым прикреплены субтитры.
  3. Если лекция содержит субтитры, скрипт находит ссылку (URL) на скачивание нужного языка (например, `English [Auto]`).
  4. Последовательно и асинхронно скачивает каждый `.vtt` файл.
  5. Объединяет скачанный текст в один большой массив данных (или склеивает в единый структурированный текст, в зависимости от логики). 
  6. Формирует конечный файл и вызывает браузерный API загрузки (`chrome.downloads.download`), сохраняя его на компьютер пользователя.

## 🚀 Пошаговый алгоритм работы (Флоу)

Вот что происходит с момента нажатия на кнопку:

1. **Инициализация:** Вы нажимаете «Скачать субтитры» в Popup.
2. **Запрос ID:** Popup отправляет сообщение в Content Script с просьбой найти `courseId` текущей страницы Udemy.
3. **Получение структуры курса:** Получив `courseId`, Popup передает его в Background Script. Background Script обращается к API Udemy, чтобы получить список всех лекций.
4. **Фильтрация:** Background Script понимает, сколько всего видеолекций нужно обойти и отправляет это число в Popup для настройки прогресс-бара.
5. **Итерация и Скачивание:** Расширение пробегается по каждой лекции. По мере скачивания VTT блоков Background Script оповещает Popup об изменении прогресса (`1/23`, `2/23` и т.д.), и полоска ползет вправо.
6. **Финал:** Все субтитры собираются в один файл (или JSON). Срабатывает триггер на загрузку файла на диск. Прогресс-бар становится 100%, и выводится статус «Успешно скачано!».

## 🛠 Установка

Так как это локальное расширение (unpacked), установка занимает всего пару кликов:

1. Откройте Google Chrome или другой браузер на базе Chromium.
2. Перейдите на страницу управления расширениями: `chrome://extensions/`.
3. Включите **«Режим разработчика»** (Developer mode) в правом верхнем углу.
4. Нажмите кнопку **«Загрузить распакованное расширение»** (Load unpacked) в левом верхнем углу.
5. Выберите папку `extension`, в которой находятся исходные файлы (`manifest.json`, `popup.js`, и т.д.).
6. Иконка расширения появится на панели браузера. Готово!

## 💡 Использование

1. Авторизуйтесь под своим аккаунтом на сайте **Udemy**.
2. Откройте любой курс (страницу просмотра лекции), к которому у вас есть доступ.
3. Нажмите на иконку расширения в панели браузера.
4. В появившемся окне нажмите большую кнопку для старта.
5. Подождите, пока заполнится прогресс-бар. Файл с субтитрами автоматически скачается в вашу папку `Загрузки`.

## ⚙️ Безопасность и Приватность

Расширение **не собирает ваши логины, пароли или личные данные**. \
Все запросы происходят исключительно локально между вашим браузером и официальным API сервером Udemy с использованием уже существующих сессионных cookie. Доступ к `*://*.udemy.com/*` необходим только для того, чтобы иметь право на чтение списка лекций от вашего (уже авторизованного) лица.
